# Aether
### Video Demo: https://youtu.be/1gVdmMYKPLY
### Description:
# Aether is an air pollution software designed to provide an overview of the Air Quality across the globe.
# Users have access to a map interface that defaults to their current location and highlights the Air Quality Index (AQI) 
# as well as a breakdown of different pollutants and their contribution to the overall air quality in that area.
